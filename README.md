# highlightjs-zig

## Description

Zig - a language grammar for highlightjs

Performance:

<img src="./image.png" width="300" height="400">

### Usage

```javascript
var hljs = require('highlightjs');
var hljsZig = require('highlightjs-zig');
require('src/zig.style.css')

hljs.registerLanguage("zig", hljsZig);
hljs.highlightAll();
```
